js 匿名函数

为什么不给函数定义名称？
因为函数名称有时候没有用
什么请看下使用匿名函数：

在对象里面定义一个函数属性的时候

var jabbaTheHut = {
 laugh : function() { alert("ho ho ho ho"); }
}
jabbaTheHut.laugh();

定义一个列表

var toDoToday = [
 function() { alert("Aren't you a little short for a storm trooper?") },
 function() { alert("Boring conversation anyway") },
];
for(var x=0; x<toDoToday.length; x++) {
 toDoToday[x]();
}

将函数作为参数调用其他函数时

// function statement
function itIsATrap(theTrap){
 theTrap();
}

// function operator
itIsATrap(function() { alert("Many Bothans died to bring us this information"); });

这种方法在定义单应用的函数时很有用，比如对象的方法或者时间处理时。
比如在jquery里，我们经常会用到

$(document).ready(function() {
    alert("page has loaded.");
});

在if条件里面定义函数

var chooseSide;
if(skywalker == "Luke Skywalker") {
    chooseSide = function() {
        return "jedi";
    }
}
else {
    chooseSide = function() {
        return "sith";
    }
}
alert(chooseSide());

在循环语句里定义函数

for(var x=0; x<=900; x++) {
    var myFunction = function() {
        alert("When " + x + " years old you reach, look as good you will not.");
    }
    myFunction();
}

box-sizing

css 属性 box-sizing
ie标准模式下，宽度、高度不包括padding和border
在兼容模式下，高度、宽度会把padding和border都算在里面，也就是有可能导致内容显示部分很小。
我们可以使用box-sizing属性来指定计算方法
box-sizing: content-box 宽度、高度仅仅是内容的高度，padding和border设置后会另加
box-sizing: border-box 宽度和高度会包括padding和border
