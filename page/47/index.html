What is Double Exclamation Marks in Javascript Mean

两个感叹号表示非非,跟没有感叹号的效果一样.

既然两个感叹号跟没有感叹号的效果一样，为什么要用两个感叹号

两个感叹号可以做类型转换，比如下面这段代码

<html>
    <head></head>
    <body>
        <script>
            var t = null;
            alert(!!t); //false
            t = '';
            alert(!!t); //false
            t = 1;
            alert(!!t); //true
            t = 0;
            alert(!!t); //false
            t = 101;
            alert(!!t); //true
            t = '111';
            alert(!!t); //true
            t = '0';
            alert(!!t); //true
        </script>
    </body>
</html>
