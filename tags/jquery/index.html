jQuery开发插件

如何添加jQuery插件，有两种方法:

静态方法

jQuery.extend(object); //为扩展jQuery类本身.为类添加新的方法。

实例方法

jQuery.fn.extend(object); //给jQuery对象添加方法。

例子:

jQuery.fn.extend

$.fn.extend({
    hilight:function(){
        // Our plugin implementation code goes here.
    }
})
我们的插件通过这样被调用：
$('#myDiv').hilight();

jQuery.extend

$.extend({
　　add:function(a,b){returna+b;}
});

$.add(3,4); //return 7
