JavaScript系统学习 第21章 Ajax与Comet

虽然现在有很多框架，对ajax也以一些不同的方式呈现，比如AngularJS里有一个服务$http。但是做为一个web开发人员，知道ajax的家底还是很有必要的。
什么是Ajax:
Ajax是一种向服务器请求额外数据而无需卸载页面的一种技术。
Ajax技术的核心是XMLHttpRequest对象（简称XHR）。
XHR为向服务器发送请求和解析服务器响应提供了流畅的接口。能够以异步方式从服务器取得更多信息，意味着用户单击后，可以不必刷新页面也能取得新数据，然后再通过DOM将新数据插入到页面中。

JavaScript系统学习 第20章 JSON

互联网上传输结构化数据是非常重要的，现在常用的是JSON。
什么是JSON:
JSON是一种数据格式，不是一种编程语言。
JSON并不从属于Javascript，并不是只有Javascript才使用JSON，很多编程语言都有针对JSON的计时器和序列化器。
通过学习，需要知道json格式的数据，如果解析与序列化。比如json格式的数据如何与string类型的数据相互转换。
