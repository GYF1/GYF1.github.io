返回顶部 Back To Top

实现“返回顶部”需要做到的地方：
“返回顶部”的标记只有当浏览的部分超过浏览器窗口的高度再显示
点击“返回顶部”效果最好能酷点，平滑过渡

一、jQuery实现

代码

1.1 “返回顶部”的标记的显示与隐藏

监听window的滚动事件，如果滚动的高度比我们设定的高度大，便显示“返回顶部”，否则隐藏。
代码

    /** 最大高度，超过了便显示“返回顶部” */
    var upperLimit = 1000;

    /** id为totop，是我们用来显示“返回顶部”的部分 */
    var scrollElem = $('#totop');

    /** 默认隐藏“返回顶部” */ 
    scrollElem.hide();

    /** 监听滚动事件 */ 
    $(window).scroll(function () {            
        var scrollTop = $(document).scrollTop();       
        if ( scrollTop > upperLimit ) {
            $(scrollElem).stop().fadeTo(300, 1);           
        }else{       
            $(scrollElem).stop().fadeTo(300, 0);
        }
    });

1.2 “返回顶部”事件响应

$(scrollElem).click(function(){
    $('html, body').animate({scrollTop:0}, scrollSpeed); return false;
});
