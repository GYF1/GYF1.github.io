AngularJS 疑难杂症

整理一些开发过程中遇到的问题，未完待续..

1. Error: ng:btstrpd App Already Bootstrapped with this Element

另一种类似的错误：WARNING: Tried to load angular more than once.

1.1 症状

页面上的东西全部重复加载了，而且每次刷新，重复的还不一样。
比较的郁闷，没有发现什么地方重复的初始化了，在google里搜，竟然没有完全同样的问题。

1.2 原因

  • 导入js要放到body结束之前
  • 页面上导入了一个jquery.js, 去掉就好了，原因应该是加载angularjs后，已经自动引入了jquery
  • 网上还有一种错误，可能是重复的引入了angularjs，这个应该比较好发现

2. ‘$’ is not defined.

1.1 症状

在directive里调用 jquery，报错

1.2 原因

项目中使用了JSHint, 需要在文件的最前面加上

/*global $:false */

3. 绑定的$scope 没反应

可能顺序有问题，比如下面两段代码的先后顺序就是固定的

$scope.uiConfig = {
    dayClick: $scope.alertEventOnDayClick
};    
$scope.alertEventOnDayClick = function( event, allDay, jsEvent, view ){
};
