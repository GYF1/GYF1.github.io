变量 var

  • 每个变量声明都要加上var关键字 *
    如果你不指定关键字var，该变量就会暴露在全局作用域（window）中，这很可能会覆盖全局作用域中的同名变量，从而引发问题（另外GC也会因此而无法有效回收内存啊），所以务必用var声明变量。
