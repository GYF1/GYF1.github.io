js 匿名函数

为什么不给函数定义名称？
因为函数名称有时候没有用
什么请看下使用匿名函数：

在对象里面定义一个函数属性的时候

var jabbaTheHut = {
 laugh : function() { alert("ho ho ho ho"); }
}
jabbaTheHut.laugh();

定义一个列表

var toDoToday = [
 function() { alert("Aren't you a little short for a storm trooper?") },
 function() { alert("Boring conversation anyway") },
];
for(var x=0; x<toDoToday.length; x++) {
 toDoToday[x]();
}

将函数作为参数调用其他函数时

// function statement
function itIsATrap(theTrap){
 theTrap();
}

// function operator
itIsATrap(function() { alert("Many Bothans died to bring us this information"); });

这种方法在定义单应用的函数时很有用，比如对象的方法或者时间处理时。
比如在jquery里，我们经常会用到

$(document).ready(function() {
    alert("page has loaded.");
});

在if条件里面定义函数

var chooseSide;
if(skywalker == "Luke Skywalker") {
    chooseSide = function() {
        return "jedi";
    }
}
else {
    chooseSide = function() {
        return "sith";
    }
}
alert(chooseSide());

在循环语句里定义函数

for(var x=0; x<=900; x++) {
    var myFunction = function() {
        alert("When " + x + " years old you reach, look as good you will not.");
    }
    myFunction();
}

闭包实践

闭包是什么，不太容易描述
但是本质上却是比较简单的

闭包可以做什么:

用来隐藏一些私有的方法

例子：

a = (function () {
    var privatefunction = function () {
        alert('hello');
    }

    return {
        publicfunction : function () {
            privatefunction();
        }
    }
})();

a是一个对象，我们可以通过a调用到publicfunction，但是没有办法调用到privatefunction
很多人一看这个例子可能会有点吃惊，因为自己早就这么使用了，只是自己不知道这就是闭包。

jQuery开发插件

如何添加jQuery插件，有两种方法:

静态方法

jQuery.extend(object); //为扩展jQuery类本身.为类添加新的方法。

实例方法

jQuery.fn.extend(object); //给jQuery对象添加方法。

例子:

jQuery.fn.extend

$.fn.extend({
    hilight:function(){
        // Our plugin implementation code goes here.
    }
})
我们的插件通过这样被调用：
$('#myDiv').hilight();
